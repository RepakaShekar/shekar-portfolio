# ⚙️ Run Project Locally

### Clone the repository
git clone https://github.com/RepakaShekar/shekar-portfolio.git


### Navigate to the project
cd shekar-portfolio


### Install dependencies
npm install


### Start development server
npm run dev

Project runs at:
http://localhost:5173
